# 생일 선물상자 게임 - 통합 기획서 (v6)

## 0. 개요

### 0.1 정보
- **플랫폼**: 웹앱 (PWA, iOS Safari 호환)
- **프레임워크**: Phaser 3
- **화면**: 세로 모드, 스마트폰 기준
- **저장**: localStorage
- **기간**: 5일 (Day 1~4 제작, Day 5 최종 제작 + 포장 + 수확)
- **실제 일정**: Day 1(토) → Day 2(일) → Day 3(월) → Day 4(화) → Day 5(수, 생일)

### 0.2 내러티브
여자친구의 24번째 생일을 기념하여, 김유찬이 자신의 비서 로봇 "콜드유(ColdU)"를 보내 여자친구가 선물을 준비하도록 돕게 한다. 콜드유는 김유찬의 지시를 받고 왔지만 선물 수령인이 누구인지는 모른다. 여자친구는 5일간 파츠를 모아 선물상자를 완성하고, 마지막 날 상자를 열면 김유찬이 준비한 증명서가 등장한다.

### 0.3 핵심 루프
1. 자원지 수확 (터치)
2. 공방에서 파츠 제작 (자원 소모 → 타이머)
3. 파츠 자동 부착 (상자로 날아감)
4. 목표 달성 → 다음 날로 이동

### 0.4 UX 원칙
- Day 1 튜토리얼에서 모든 시스템 학습 (콜드유가 안내)
- Day 2~4는 UI 알림·뱃지로 자율 학습 유도 + 콜드유 간단 안내 메시지
- Day 5: 최종 제작 + 포장 + 생일 시퀀스 + 엔딩
- 모든 새 요소는 `NEW` 뱃지 + 팝업 1회로 알림

## 1. 시스템

### 1.1 자원 (5종)

| ID | 이름 | 생산 주기 | 사이클당 생산량 | 저장 한도 | 사용 가능 Day |
|----|------|---------|-------------|---------|---------|
| wood | 나무 | 1.5시간 | 120 | 960 | 1 |
| flower | 꽃 | 2시간 | 80 | 640 | 2 |
| stone | 돌 | 1.5시간 | 150 | 1,200 | 2 |
| metal | 금속 | 3시간 | 300 | 1,200 | 3 |
| gem | 보석 | 3시간 | 250 | 1,000 | 4 |

- **밸런싱 기준**: 하루 2회 수확 전제 (12시간 간격). 3회 수확은 드물게 열심히 하는 경우.
- **저장 한도**: 12시간 생산분 기준 (한도 도달 시 사이클 정지).
- **수확 로직**: 수확 시 쌓인 자원 전부 인벤토리로 이동 + 다음 사이클 자동 시작.
- **자원지**: 해당 Day 진입 시 건설 메뉴에 항목 추가.
- **수급 잉여율**: 약 20~30% (편안한 난이도).

### 1.2 건물 (8종)

| ID | 이름 | 역할 | 사용 가능 Day | 준비 시점 | 건설 비용 |
|----|------|-----|---------|----------|---------|
| box | 선물상자 | 파츠 수집 | 1 | 처음부터 | 무료 (시작 지급) |
| wood_farm | 나무밭 | 자원 생산 | 1 | 처음부터 | 무료 (시작 지급) |
| woodshop | 목공방 | 파츠 제작 | 1 | 처음부터 | wood 500 |
| flower_farm | 꽃밭 | 자원 생산 | 2 | 아침에 바로 | wood 1,000 |
| quarry | 채석장 | 자원 생산 | 2 | 오후에 추가 | wood 1,000 |
| mine | 광산 | 자원 생산 | 3 | 아침에 바로 | stone 1,000 |
| jewelshop | 세공소 | 파츠 제작 | 3 | 오후에 추가 | wood 1,500 + stone 1,000 |
| gem_cave | 수정동굴 | 자원 생산 | 4 | 아침에 바로 | stone 1,500 + metal 500 |

- **건물 개수**: 각 1채만 건설 가능 (업그레이드·추가 건설 없음)
- **건설 시간**: 즉시 완료
- **건설 위치**: 빈 칸에 자유 배치 가능
- **상자**: 맵 중앙 2x2 고정, 이동·삭제 불가
- **시차 준비**: Day 2와 Day 3에서 2번째 건물은 시작 2~3시간 후 자동으로 건설 가능 (콜드유 알림)

### 1.3 공방 (2종)

| ID | 이름 | 제작 가능 | 동시 제작 |
|----|------|---------|---------|
| woodshop | 목공방 | 나무·꽃·돌 기반 파츠 | 1 |
| jewelshop | 세공소 | 금속·보석 기반 파츠 | 1 |

- **제작 플로우**: 공방 터치 → 제작 메뉴 → 파츠 선택 → 자원 소모 → 타이머 시작
- **타이머 완료 시**: 파츠가 자동으로 상자로 날아가서 부착

### 1.4 파츠 (24개)

- **총 개수**: 24
- **Day별 분배**: 4-5-5-5-5
- **부착 방식**: 제작 완료 시 자동 (유저 조작 불필요)
- **상세 목록**: 부록 A 참조

### 1.5 선물상자 (중앙 고정 오브젝트)

- **초기 상태**: 비어있음 (형태 없음)
- **시각 단계**: 7단계 (파츠 부착 진행에 따라 변화)

| 단계 | 진행도 | 상자 외형 |
|------|-------|----------|
| 1 | 0/24 | 빈 공간 (윤곽선만) |
| 2 | 1~4/24 | 나무 골격 형성 |
| 3 | 5~9/24 | 상자 형태 완성 + 꽃·숫자 장식 |
| 4 | 10~14/24 | 받침대 + 금속 장식 추가 |
| 5 | 15~19/24 | 보석·왕관 장식 추가 |
| 6 | 20~23/24 | 거의 완성 (리본·배너 추가) |
| 7 | 24/24 | 최종 완성 (뚜껑 매듭 부착) |

- **파츠 부착**: 파츠 제작 완료 시 자동 부착 애니메이션 재생 (2초)
- **24번째 파츠 부착 시**: 포장 타이머 시작 (90분)
- **포장 완료 시**: 수확 가능 상태 전환
- **수확 시**: 엔딩 시퀀스 재생

### 1.6 콜드유 (NPC)

| 항목 | 내용 |
|------|-----|
| 정체 | 김유찬의 비서 로봇 |
| 등장 Day | Day 1 (튜토리얼), Day 2~4 (간단 안내), Day 5 (생일 시퀀스 + 엔딩) |
| 말투 | 존댓말, 감정 없음, 사무적·기능적 표현 |
| 표현 | 맵 고정 스프라이트 + 화면 하단 대화창 UI |
| 대화 내용 | 부록 C 대화 스크립트 참조 |
| 에셋 | 맵 스프라이트 1종, 대화창 아이콘 2~3종 |
| 애니메이션 | 없음 (정적 이미지) |

### 1.7 UI 구성 (메인 화면)

**상단 바**
- 현재 Day 표시
- 자원 인벤토리 (5종, 미사용 자원은 자물쇠)
- 파츠 진행도 바 (X/24)

**하단 바**
- 건설 버튼 (+ NEW 뱃지)
- 공방 버튼 (+ 제작중 알림)
- 상자 버튼 (바로가기)

**맵**
- 스와이프 이동
- 건물 터치 시 상호작용
- 자원지 수확 가능 시 `!` 이펙트

### 1.8 타이머·저장

**타이머**
- 실시간 기반 (앱 종료 중에도 진행)
- 앱 재진입 시 경과 시간 반영
- 관리 대상: 자원지 생산, 파츠 제작, 포장 타이머, 시차 타이머

**저장 (localStorage)**
- 자원 보유량
- 건물 건설 상태·위치·타이머
- 파츠 제작·부착 상태
- 현재 Day
- 튜토리얼 진행 상태
- 포장 타이머
- 시차 타이머 (Day 2 채석장, Day 3 세공소)
- 수확 여부

## 2. Day 진행 시스템

### 2.1 Day 전환 조건
- 당일 목표 파츠 개수 완성 **AND** 실제 날짜 변경
- 조건 미충족 시 다음 Day로 넘어가지 않음

### 2.2 Day 진입 시 자동 발생
- 콜드유 안내 메시지 (대화창)
- 새 건물·자원 사용 가능 및 UI 업데이트 (`NEW` 뱃지)
- 당일 목표 표시

### 2.3 Day 완료 조건
- 당일 목표 파츠 X개 모두 부착 완료

### 2.4 Day 완료 시 자동 발생
- 진행도 바 업데이트
- 조용히 다음 날 대기 상태로 전환 (실제 날짜 변경 대기)

### 2.5 시차 시스템

Day 2와 Day 3에서 2번째 건물은 바로 건설할 수 없고, 시작 2~3시간 후에 건설 가능해진다.

| Day | 바로 건설 가능 | 오후에 추가 (2~3시간 후) |
|-----|----------|---------------------|
| 2 | 꽃밭 | 채석장 |
| 3 | 광산 | 세공소 |

**시차 구현:**
- Day 진입 시 시차 타이머 시작 (2.5시간 = 150분 기본값)
- 타이머 완료 시:
  - 콜드유 알림: "채석장을 만들 수 있게 되었습니다." (또는 세공소)
  - 건설 메뉴에 해당 건물 추가 (`NEW` 뱃지)
  - 앱 비활성 중이면 재진입 시 즉시 반영
